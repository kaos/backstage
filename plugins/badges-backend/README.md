# Badges Backend

Backend plugin for serving badges. Default implementation uses
[badge-maker](https://www.npmjs.com/package/badge-maker) for creating
the badges, in SVG.

## Setup

The list of all badges to offer are passed to the badges-backend
`createRouter()`.
